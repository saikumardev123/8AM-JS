JS Operators:

Arithmetic Operators:

 +,-,*,/(quotient),% (remainder), ++, --

 <html>

<head>

</head>

<body>
    <script>
        var a = 10;
        console.log(a++); // prints 10, a= 10+1 = 11
        console.log(++a); // a= 11 + 1 => 12, prints 12
        console.log(--a); // 11, prints 11
        console.log(a--); //  prints 11 and then a= 11-1 = 10
        console.log(a);  // prints 10
    </script>
</body>
